# invalid

结构体字段校验

## 支持规则

| 规则 | 语法 | 说明 | 示例 |
| :-- | :--: | :-- | :-- |
| 非 | ! | 字段不可等于所给值 | !nil<非空> |
| 枚举 | {} | 字段需被包含在枚举值中 | {'1', "2"} |
| 范围 | `[` 或 `(` 、 `]` 或 `)` | 数值或时间的范围 | [0, )<非负> |
| 正则 | r'' | 正则匹配 | r'[\u4E00-\u9FA5]'<包含中文> |
| 默认值 | @... | 字段默认值 | @"1"<字符串1> |
| 引用 | @<字段名> | 指代结构体中的某个字段值 | @ID |

## 规则详细说明

### 非

如果字段类型为指针
